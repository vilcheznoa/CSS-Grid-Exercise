# CSS Grid Exercise
Galería de imágenes de una boda utilizando **CSS Grid** para organizar un conjunto de 11 fotografías en un layout responsivo y limpio.

## Características
- Distribución en cuadrícula con `grid-template-columns: repeat(4, 1fr)`.
- Filas automáticas con altura flexible mediante `grid-auto-rows: minmax(150px, auto)`.
- Imágenes con `object-fit: contain` para mostrarse completas sin recortes.
- Texto accesible mediante descripciones en atributos `alt`.
- Fuente personalizada cargada con `@font-face` (Glitter).
